[Zola](https://www.getzola.org/) でページを作ってみました。  
テーマは [kita](https://github.com/st1020/kita) を使用しました。

GitHub Actions Workflow の参考です。ほぼデフォルトで、gh-pages ブランチにビルド成果物が入り、それを公開しています。  
なので、公開パスは `gh-pages` の `/(root)` となります。GitHub で直接設定している箇所はそれ以外ありません。

```yaml: gh-pages.yml
on: push
name: Build and deploy GH Pages
jobs:
  build:
    permissions:
      contents: write
    runs-on: ubuntu-latest
    if: github.ref == 'refs/heads/main'
    steps:
      - name: checkout
        uses: actions/checkout@v6
        with:
          submodules: true
      - name: build_and_deploy
        uses: shalzz/zola-deploy-action@master
        env:
          TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

設定を確かめて遊ぶ目的なので大した内容が増える見込みはありません。  
メモ置場になる予定です。
